=Ö Advanced TypeScript Features

Static Methods & Properties

Static members belong to the class itself, not instances. You can call ContactFormValidator.validateEmail() without creating a new instance.

Utility Types

  • Partial<T> - Makes all properties optional
  • Readonly<T> - Makes all properties readonly
  • Pick<T, K> - Pick specific properties
  • Omit<T, K> - Omit specific properties

Type Guards

Type guards help TypeScript understand what type a variable is at runtime. The if (textarea && counter) checks ensure elements exist before using them.

Immutability

Using Object.freeze() and Readonly types prevents accidental modification of data, making code more predictable and safer.
